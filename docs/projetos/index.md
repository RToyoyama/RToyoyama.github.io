## Engenharia de Dados

### [Currency ETL Pipeline](currency-etl-pipeline.md)

![Python](https://img.shields.io/badge/Python-3.13-blue?logo=python&logoColor=white)
![Airflow](https://img.shields.io/badge/Apache%20Airflow-2.9-017CEE?logo=apacheairflow&logoColor=white)
![PostgreSQL](https://img.shields.io/badge/PostgreSQL-15-336791?logo=postgresql&logoColor=white)
![Docker](https://img.shields.io/badge/Docker-Compose-2496ED?logo=docker&logoColor=white)
![CI](https://img.shields.io/badge/CI%2FCD-GitHub%20Actions-2088FF?logo=githubactions&logoColor=white)

Pipeline ETL automatizado que coleta cotações de USD, EUR e BTC em tempo real,
transforma com Pandas e persiste em PostgreSQL. Orquestrado com Airflow em
execução horária, containerizado com Docker e com CI/CD via GitHub Actions.
